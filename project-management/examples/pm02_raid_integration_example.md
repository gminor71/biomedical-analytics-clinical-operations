# PM02 RAID Integration Example

This example shows how a project item can be triaged with PM02 and then
automatically appended to the working RAID log.

---

## Input Item

Example PM02 input YAML:

```yaml
project_name: Phase 2 oncology interim analysis support
item_title: Delayed site query resolution
item_description: High-enrolling sites have not resolved outstanding efficacy data queries, creating risk to interim dataset readiness.
project_phase: execution
functional_area: cross_functional
regulatory_environment: high
timeline_impact: high
deliverable_impact: high
issue_is_current: true
decision_needed: false
vendor_related: false
primary_function: data_management
