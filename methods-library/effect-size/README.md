# Effect Size Reference

## Overview

This folder contains reference material for interpreting and selecting
effect size measures across different types of endpoints.

Effect sizes are critical for:

* Interpreting clinical relevance
* Comparing treatment effects
* Supporting SAP development and reporting

---

## Categories

### Binary Endpoints

* Risk difference
* Risk ratio
* Odds ratio

### Time-to-Event Endpoints

* Hazard ratio
* Restricted Mean Survival Time (RMST)

### Continuous Endpoints (future)

* Mean difference
* Standardized mean difference

---

| Agent | Effect Size Usage |

---

## Key Principle

> Effect size selection should align with both statistical validity and 
clinical interpretability.
