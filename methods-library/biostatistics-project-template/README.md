# Biostatistics Project Template

Template structure used for clinical trial statistical analysis projects.

Provides a standardized layout for:
- data organization
- TFL development
- analysis scripts
- reproducible reporting

Used as a starting structure for CT01–CT04 workflows.

biostatistics-project-template/

├── data/       # Raw and processed datasets

├── figures/    # Individual Figures

├── notebooks/  # For Future Use

├── results/    # Individual Outputs, Diagnostics, and Output Packets (TLFs)

├── rmd/        # Statistical Analysis Plan and TLF Shells

├── src/        # Analysis code

├── tables/     # Individual Tables
