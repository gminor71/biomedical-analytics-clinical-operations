# Trial Context

This folder contains **stable trial-level reference documents** used across all simulation phases.

These files define the operational and clinical framework of Master Trial 01 and are intended to remain largely unchanged as scenarios evolve.

---

## Contents

Typical documents include:

* `protocol_synopsis.md`
* `visit_schedule.md`
* `endpoint_summary.md`
* `trial_overview.md`

---

## Purpose

The trial context represents:

* protocol-level expectations
* visit and assessment structure
* endpoint definitions
* overall study design

These files provide background and reference information for scenario development and CRA decision-making.

---

## Working Guidance

* Treat this folder as the “rules of the trial.”
* Scenario-specific events should **not** be added here.
* Operational changes over time should be represented in the `timeline/` folder.
